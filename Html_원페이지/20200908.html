저는 개발자가 되기 위해 다양한 역량을 강화하였습니다.

첫째, 다양한 프로그래밍 언어를 공부하며 프로그래밍 기반을 다졌습니다. JAVA를 기반으로 웹 개발에 필요한 Oracle, MySQL, HTML, CSS, JSP, Java Script, Spring framework, 등의 다양한 언어를 경험해보았으며 소규모 프로젝트에 직접 참여하여 응용하였고, 정보처리산업기사 자격증을 취득하여 소프트웨어 개발 직무를 수행할 수 있는 기본 실력이 갖추어졌음을 증명하였습니다.

둘째, 학창시절과 다양한 사회 프로젝트를 진행하며 팀워크를 강화했습니다. 정확하고 진실한 정보를 가지고 팀원들과 협업을 하면 업무적 효율이 높아진다고 생각합니다, 처음 진행한 팀 프로젝트에서는 팀장으로써, 개발과 퍼블리싱을 분리해 진행하며 팀원과 갈등 해결과 의견을 조율하였습니다.

셋째, 기록하는 개발자 웹 프로그래밍을 공부하고부터 , 직접 운영하는 블로그에 개발하면서 느낀 점과 자주 하는 실수 등을 기록하였습니다. 처음에는 404, 500번 에러가 뜨면 무조건 교수님의 도움을 받았지만, 에러를 기록하고, 스스로 해결하고자 하면서 코드를 이해할 수 있었고 프로젝트에 응용하여 완성도를 높일 수 있었습니다.
